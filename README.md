## Run anywhere

Clone the project

```bash
  git clone https://github.com/witczakxd/Snake-Multiplayer-Client
```

Go to the project directory

```bash
  cd Snake-Multiplayer-Client
```

Install requirements

```bash
  pip install requirements.txt
```

Run the game

```bash
  python main.py
```
